HTML & CSS Cheat Sheet

HTML Structure & Key Concepts

Concept Syntax / Tag Description
Element<h1>Heading</h1>HTML element is a unit of content composed of tags and content.
Opening Tag<p>Starts an HTML element.
Closing Tag</p>Ends an HTML element.
Self-Closing Tag<br>Does not require a closing tag (used for line breaks).
Attributeid="main"Provides additional information about an element.
HyperText<a href="..."></a>Links to other content or pages.

Display & Style Text

Element Syntax Description
Paragraph<p>Text</p>Defines a paragraph block of text.
Span<span>Inline text</span>Used for styling inline portions of text.
Emphasis<em>important</em>Renders text in italics for emphasis.
Strong<strong>bold</strong>Displays text as bold, indicating importance.

Lists

Type Syntax Description
Unordered List<ul><li>Item</li></ul>Creates a bulleted list.
Ordered List<ol><li>Item</li></ol>Creates a numbered list.

Images and Media

Element Example Description
Image<img src="image.jpg" alt="desc">Displays an image; requires src and alt attributes.
Video<video src="video.mp4" controls></video>Embeds a video with playback controls.
Audio<audio controls><source src="song.mp3"></audio>Embeds audio with controls.
Embed<embed src="file.gif">Embeds external media like GIFs or PDFs.

Semantic HTML

Element Example Description
<header>Page top sectionIntroductory content or site branding.
<main>Wraps main contentHolds the primary page content.
<footer>Page bottomContains copyright or links.
<section>Logical content groupingOrganizes related content.
<article>Blog post or self-contained blockCan stand alone outside the page.
<aside>SidebarContains additional, related content.
<figure><figure><img src="..."></figure>Wraps media like images or charts.
<figcaption>Text inside <figure>Describes the media.

HTML Tables

Element Syntax Description
Table<table>Creates a table container.
Row<tr>Defines a row of cells.
Cell<td>Defines a cell with data.
Header Cell<th>Table header text.
colspan / rowspan<td colspan="2">Spans columns/rows.
<thead>, <tbody>, <tfoot>Sections of tableSeparates header, body, and footer rows.

Common Form Input Types

Type Syntax Description
Text <input type="text"> Standard single-line text field.
Password <input type="password"> Hides characters as the user types.
Email <input type="email"> Validates input for an email format.
Number <input type="number"> Accepts numeric input (with min/max options).
Checkbox <input type="checkbox"> Creates a checkable box.
Radio <input type="radio" name="option"> Selectable options where only one can be chosen per group.
Submit <input type="submit" value="Submit"> Submits the form data.

Concept Example Description
DOCTYPE<!DOCTYPE html>Declares the HTML version (HTML5).
html element<html lang="en">Root element for the document.
head element<head></head>Contains metadata, title, and links.

CSS Basics

Property Example Description
color color: red; Sets the text color.
background-color background-color: lightblue; Sets the background color of an element.
font-family font-family: Arial, sans-serif; Specifies the font style for text.
font-size font-size: 16px; Sets the size of the text.
margin margin: 20px; Sets outer spacing around elements.
padding padding: 10px; Sets inner spacing inside elements.
border border: 1px solid black; Draws a border around elements.
display display: flex; Defines how elements are displayed (block, inline, flex, etc.).
position position: relative; Controls element positioning (static, relative, absolute, fixed).
text-align text-align: center; Aligns text horizontally (left, center, right).

CSS for UI & UX Design

Property Example Purpose
box-shadow box-shadow: 0 2px 6px rgba(0,0,0,0.2); Adds soft depth/shadow to elements (like cards, buttons).
border-radius border-radius: 8px; Creates rounded corners for smoother UI elements.
transition transition: all 0.3s ease; Makes hover/animation effects smoother.
hover selector button:hover { background-color: navy; } Changes style when user hovers for better feedback.
cursor cursor: pointer; Makes clickable elements feel interactive.
flexbox display: flex;
justify-content: center;		 Layouts content easily across rows/columns.
gap gap: 20px; Controls spacing between flex/grid items.
z-index z-index: 10; Controls element stacking order (for modals/tooltips).
overflow overflow: auto; Handles scrollbars when content overflows.

Display Values

Value Example Description
block display: block; Starts on a new line, takes full width (e.g., div, p).
inline display: inline; Stays within line, doesn't break layout (e.g., span).
inline-block display: inline-block; Like inline but supports width/height.
flex display: flex; One-dimensional layout (row or column).
grid display: grid; Two-dimensional layout (rows + columns).
none display: none; Hides the element from the page.

CSS Flexbox

Property Example Description
display display: flex; Enables flexbox on a container element.
flex-direction flex-direction: row; Sets the direction (row, row-reverse, column, column-reverse).
justify-content justify-content: space-between; Aligns items on the main axis (start, center, space-between, etc.).
align-items align-items: center; Aligns items on the cross axis (start, center, stretch, etc.).
gap gap: 10px; Sets spacing between flex items.
flex-wrap flex-wrap: wrap; Allows items to wrap onto multiple lines.
align-self align-self: flex-end; Overrides align-items for a single item.
flex flex: 1; Sets item size relative to others.

CSS Grid

Property Example Description
display display: grid; Enables grid layout on a container.
grid-template-columns grid-template-columns: 1fr 1fr; Defines the number and width of columns.
grid-template-rows grid-template-rows: 100px auto; Defines the number and height of rows.
gap gap: 20px; Sets space between rows and columns.
grid-column grid-column: 1 / 3; Item spans across columns.
grid-row grid-row: 1 / 2; Item spans across rows.
place-items place-items: center; Aligns items both vertically and horizontally.
repeat() grid-template-columns: repeat(3, 1fr); Repeats columns automatically.
